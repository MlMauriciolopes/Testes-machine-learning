#Exercício: Criar um programa que o usuário digite o número e o programan calcule o número fatorial dele
#4! = 4*3*2*1 (fatorial)
numero = int(input('Digite um número: '))
fatorial = numero
contador = 1
while (numero-contador)>1: #Enquanto o número que a pessoa informou, menos o contador for maior que 1, a função vai ser executada
    fatorial = fatorial*(numero-contador) 
    contador += 1
print('{0}! = {1}'.format(numero,fatorial))
